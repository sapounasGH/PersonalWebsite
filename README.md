# Personal Website — Christos Sapounas

First university project.
A personal website featuring a multi-page layout, animated background, interactive image effects, and links to social media profiles.

---

## Project Structure

```
PersonalWebsite/
├── homepage.html       # Landing page with intro and animated background
├── abtme.html          # About Me page
├── prjct.html          # Projects page
├── hb.html             # Hobbies & Miscellaneous page
├── cont.html           # Contact page
└── personal website/   # Additional assets folder
```

---

## Features

- **Multi-page navigation** — Consistent navbar across all pages linking to Home, About Me, Projects, M&M (Hobbies), and Contact.
- **Animated background** — A looping GIF background with a START/STOP toggle button so the user can pause the animation.
- **Interactive image** — A profile photo that swaps to a different image on hover using JavaScript.
- **Social media links** — Quick-access icons in the navbar for Instagram, Spotify, and Twitter/X.
- **Custom font** — Uses the *Audiowide* font from Google Fonts for a distinctive look.
- **Fully static** — No frameworks or build tools required; open any `.html` file directly in a browser.

---

## Technologies Used

- **HTML5**
- **CSS3** (inline styles)
- **Vanilla JavaScript** (background toggle, image hover effect)
- **Google Fonts** — Audiowide
